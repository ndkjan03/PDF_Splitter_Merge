# **CLI sẽ hoạt động như sau**

##### **1. Split toàn bộ**

`pdf split report.pdf`

##### **2. Split theo trang**

`pdf split report.pdf "1-3,5,8-10"`

##### **3. Merge nhiều file**

`pdf merge *.pdf`

##### **4. Merge thư mục**

`pdf merge split_output_20260626_210512`

##### **5. Merge đệ quy**

`pdf merge -r Documents`

##### **6. Info**

`pdf info report.pdf`
